# Starter pack commands

## Turborepo - NextJs - Chakra UI 


### Turborepo

```sh
pnpm dlx create-turbo@latest
```

### NextJS 
```
cd apps/app
pnpm dlx create-next-app@latest
```

### Frontend libraries

Chakra UI
```sh
pnpm i @chakra-ui/react @emotion/react
```
Lucide-react
```sh
pnpm i lucide-react
```
numbro 
```sh
pnpm i numbro
```
Supabase client
```sh
pnpm i @supabase/supabase-js
```
